# sh_tools
These are mini tools for monitoring linux, mysql.

# Description
Please set your sever's cron system.

[df_check]  
 you will recieve alert mail form the server if linux capasity exceeds your setting.  

[my_dump]  
 Backup mysql db and make encryption zip file.  

[replicate_check]  
 you will recieve alert mail form the server if mysql replication is inactive.  
 *set slave server's cron
